This project was developed as part of my SKL Internship to manage and monitor garment production workflows efficiently. The system focuses on order management, task allocation, milestone tracking, and status visualization for a garment export company.

Project Overview
The application helps administrators and merchandisers track complete order lifecycles — from order creation to final shipment — while monitoring department-wise tasks such as fabric, cutting, printing, sewing, and finishing.

Features
Admin and User (Merchandiser) role-based login
Order creation and management
Task allocation for different production stages
Team member assignment and tracking
Scheduled vs actual date comparison
Automatic status updates (Completed / Incomplete)
Visual status indicators (Green for on-time, Red for delayed)
Dashboard view for weekly and department-wise tasks
PDF report generation for orders and work details

 Roles & Access
Admin
Create and manage orders
Assign tasks and team members
View all dashboards and reports
User (Merchandiser)
View assigned tasks only
Update task status and completion dates
Access team details and reports

Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Tools: XAMPP, phpMyAdmin
